# LogTail 项目总结

## 项目概述

LogTail 是一个基于 Web 的日志实时监控工具，可以让您像使用 `tail -f` 命令一样，通过浏览器实时查看远程服务器上的日志文件。

```

├── server.py               # 后端服务（Python + aiohttp）

├── config.example.json     # 配置文件模板
```

## 使用方法

### 1. 配置服务器
编辑 `config.json`，填入您的服务器信息：
```json
{
  "servers": [
    {
      "host": "192.168.0.10",
      "username": "root",
      "password": "",
      "base_path": "/data/logs"
    }
  ]
}
```

### 2. 启动服务
```bash
# Windows 用户
start.bat

# 或手动启动
python server.py
```

### 3. 访问应用
打开浏览器访问：http://localhost:8080

### 4. 查看日志
1. 选择服务器
2. 浏览目录找到日志文件
3. 点击文件
4. 点击"开始监控"

## API 接口

### GET /api/servers
获取服务器列表

### GET /api/files/list
获取目录文件列表
- 参数：`server` (服务器 IP), `path` (目录路径)

### GET /api/tail
WebSocket 接口，实时查看日志
- 参数：`server` (服务器 IP), `file` (文件路径)

## 安全建议

### 生产环境部署
1. **使用 SSH 密钥认证** - 比密码更安全
2. **配置 HTTPS** - 加密通信
3. **添加用户认证** - 防止未授权访问
4. **限制访问 IP** - 设置 IP 白名单
5. **最小权限原则** - SSH 用户只给予必要权限

### 密钥认证配置
1. 生成 SSH 密钥：
   ```bash
   ssh-keygen -t rsa -b 4096
   ```

2. 复制公钥到服务器：
   ```bash
   ssh-copy-id root@192.168.0.10
   ```

3. 修改 `config.json`：
   ```json
   {
     "username": "root",
     "key_file": "C:/Users/YourName/.ssh/id_rsa"
   }
   ```

## 性能优化

1. **连接池** - 复用 SSH 连接
2. **异步 IO** - 非阻塞操作
3. **按需加载** - 目录树懒加载
4. **日志限制** - 限制显示行数避免内存溢出
